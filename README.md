Data and code for experimental results for

"Measuring the higher-order homophily in a general social hypergraph"

Yunping Wang, Mingwei Li, Dengyi Zhao and Xingqin Qi

For example, for the Congress dataset, first run `congress_pro.py`, then run `congress_plt.py`. The resulting figures and interval ratio data are stored in the corresponding folder under `figure/congress`.

The data used in this project are mainly sourced from https://github.com/nveldt/HypergraphHomophily/tree/main. Among them, Cont-hosp and hosp-DAWN come from https://github.com/arnabsarker/Higher-Order-Homophily-Simplicial-Complexes.
